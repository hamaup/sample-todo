# TODO App (TDD)

Test-Driven Developmentで作成するシンプルなTODOアプリケーション。

## セットアップ

```bash
npm install
```

## テスト実行

```bash
# テストを実行
npm test

# ウォッチモードでテスト
npm run test:watch

# カバレッジ付きテスト
npm run test:coverage
```

## 開発フロー

1. 失敗するテストを書く (RED)
2. テストを通す最小限のコードを書く (GREEN)
3. コードをリファクタリング (REFACTOR)
